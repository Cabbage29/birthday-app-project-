# Lab Exercise 2 – Report Guide

## Objective

To enhance a Java Android BirthdayApp that stores and manages birthday information using an Android Content Provider.

## Improvements implemented

1. Added a Reset button to clear the name, birthday, validation messages, and displayed list.
2. Added input validation so blank names and birthdays cannot be saved.
3. Added a date picker that formats dates as DD/MM/YYYY and prevents future dates.
4. Added a confirmation dialog before deleting all records.
5. Added a saved-record counter.
6. Improved the user interface with a purple theme, cards, rounded input fields, and clearer button grouping.
7. Sorted displayed birthday records alphabetically by name.
